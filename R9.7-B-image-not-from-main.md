## Most likely cause

`docker compose pull` pulled an old image from the registry that predates the merge. Then `--force-recreate` swapped containers but kept the wrong image. The "Downloaded newer image" message reflects newer-than-what-was-cached, not newer-than-main.

Check with:

```sh
docker compose ps --format json | jq '.[].Image'
docker image inspect <image>:<tag> | jq '.[0].Config.Labels'
docker image inspect <image>:<tag> | jq '.[0].Created'
```

You want `Created` to be AFTER 21:32 UTC. If it's before, the registry has not been pushed to since the merge.

## Fix sequence (server-side)

```sh
cd /path/to/homelabarr-ce
git fetch origin
git checkout main
git pull --ff-only origin main
git rev-parse HEAD
# MUST print 3a5c75b9967819561edd244b47cbf764eeff5721

# Build LOCALLY from the checked-out tree (do NOT pull from registry)
docker compose build --no-cache --pull backend
# the --pull here is for base images (node:24-alpine etc), not for the backend image

# Confirm the new image
docker images | grep homelabarr | head
# the topmost line should show a CREATED timestamp of seconds ago

# Stop, remove, recreate — do NOT use `pull` again
docker compose up -d --force-recreate --no-deps backend

# Confirm code inside the new container
docker exec <backend-container-name> sh -c "grep -n \"app.get('/health/detail'\" /app/server/index.js"
# expected: 824:app.get('/health/detail', requireAuth, requireRole('admin'), ...
docker exec <backend-container-name> sh -c "grep -n \"app.get('/_routes'\" /app/server/index.js"
# expected: 1221:app.get('/_routes', requireAuth, ...
docker exec <backend-container-name> wc -l /app/server/index.js
# expected: 5310 /app/server/index.js
```

If `docker exec ... grep` does NOT print the middleware, the image was not rebuilt from the checked-out source. In that case:

```sh
# nuclear option — rebuild from a clean checkout
docker compose down backend
docker rmi $(docker images -q 'homelabarr*')
git status   # confirm clean working tree at main 3a5c75b
docker compose build --no-cache backend
docker compose up -d backend
```
