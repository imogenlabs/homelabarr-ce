## Cause

Whatever orchestrator runs your deploy didn't trigger off the main push. Three normal possibilities:

1. Watchtower / git-pull cron hasn't ticked yet (next interval not reached)
2. The deploy script needs to be run manually (no automatic pull-on-push)
3. The build ran but used cached layers and produced the same image hash → container didn't actually swap

# force a real rebuild (no cache, recreate container)
